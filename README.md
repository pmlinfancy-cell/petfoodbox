## 📖 项目简介

**猫食盒子** 是一套面向校园与社区的 **户外智能投食系统**，由 **硬件终端 + 小程序前端 + 后台管理系统** 构成。  
用户通过小程序选择投喂设备、支付少量费用（如1元/20mg猫粮），即可远程触发户外投食机出粮，并通过 **猫脸识别** 记录是哪只猫吃到了猫粮，形成可追踪、可复盘、有情感连接的投喂闭环。

项目源于深大校园记忆，依托深大黑客松发起，希望用技术改善流浪猫的生存现状，同时维护校园生态平衡。

## ✨ 核心功能

### 用户端（小程序 / Demo）

- 🗺️ **校园地图**：展示深大校园内设备点位、在线状态、投喂入口
- 💰 **极速投喂**：3步内完成支付（1元/20mg起），支持套餐选择
- 📹 **投喂记录**：查看订单状态（队列中 → 已出粮 → 已进食），回传视频/截图
- 🐱 **猫猫档案**：为识别到的猫咪命名、查看年龄/品种/近期投喂历史
- 🔐 **微信授权登录**：保存个人投喂记录与猫咪互动

### 硬件端

- 🌧️ **户外耐用**：防雨、防晒、防咬、防尘设计
- ⚙️ **精准出粮**：按订单量自动出粮，支持远程控制与自检
- 📡 **视频回传**：每次投喂生成短视频或图片证据
- 🔄 **状态同步**：在线/离线监控，断网重连与补偿队列

### 后台管理系统

- 📦 **订单管理**：全订单状态流转、异常处理、退款
- 🐈 **猫脸训练库**：管理样本、识别版本、人工审核入库
- 🥣 **猫粮批次**：库存、临期提醒、更换记录
- 🛠️ **设备管理**：设备列表、固件版本、远程维护
- 📊 **运营配置**：套餐开关、轮播图、地图点位

---

## 🧠 技术亮点

- **猫脸识别**：基于开源方案（如 sysucats/cat-face）自训练模型，高置信自动入库 + 低置信人工审核
- **状态机驱动**：`已创建 → 已出粮 → 已进食`，任意阶段可转入异常并通知运营
- **人机协同闭环**：动态调控投喂量，避免过度聚集或生态失衡
- **模块化解耦**：前端可先运行 HTML Demo，后续平滑迁移至小程序与真实支付

## 🚀 快速开始（Demo 体验）

> 当前为 Phase 1 Demo，模拟完整投喂流程

1. 访问在线演示：[https://petfoodbox.lovable.app](https://petfoodbox.lovable.app)
2. 浏览深大校园地图，点击任一设备上的 **「开始投食」**
3. 选择套餐（1元/20mg 可用），点击 **「立即投喂」**
4. 支付成功，查看投喂记录与状态流转
5. 为识别到的猫咪命名，进入猫猫档案

> 完整版需配合硬件与真实支付，欢迎关注后续开源硬件方案。
